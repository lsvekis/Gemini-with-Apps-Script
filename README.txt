Apps Script with Gemini — Quick Start Guide Code
===============================================

This folder contains the example projects from the "Apps Script with Gemini — Quick Start Guide".
Each subfolder is a separate Apps Script project you can copy into the online editor or upload to GitHub.

01-core-logger
  - Basic helper + testGeminiSimple() logger demo.

02-sheets-custom-function
  - Helper + GEMINI_COMPLETE() custom function for Google Sheets.

03-sheets-sidebar
  - Helper + sidebar backend (Code.gs) and Sidebar.html for the "Gemini Prompt Pad".

04-docs-summarizer
  - Helper + onOpen() menu and summarizeSelectionWithGemini() for Google Docs.

05-gmail-draft-reply
  - Helper + draftReplyWithGemini() example using GmailApp.

06-mini-examples
  - Helper + several small practice functions (testGeminiBullets, explainCode, generateQuizQuestions, safeGeminiExample).

Created for quick GitHub upload and copy/paste into Apps Script.
